# Página Web Personal — Santiago Bastidas

Proyecto de presentación personal desarrollado en dos versiones: **sin framework (MPA)** y **con framework Bootstrap (SPA)**.

---

## Tecnologías usadas

**Versión sin framework:**
- HTML5 semántico
- CSS3 puro
- JavaScript nativo

**Versión con framework:**
- HTML5
- Bootstrap 5.3
- CSS3 personalizado
- JavaScript nativo

---

## Cómo abrir el proyecto

1. Descomprime el archivo `.zip`
2. Abre la carpeta de la versión que deseas ver
3. Abre el archivo `index.html` en cualquier navegador moderno (Chrome, Firefox, Edge)
4. No requiere servidor ni instalación adicional
